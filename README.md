# Telegram Бот для СК «Алекс»

## Описание
Telegram бот для управления спортивным клубом «Алекс» с функциями регистрации, оплаты, записи на тренировки и администрирования.

## Функции
- 🔐 Регистрация спортсменов и родителей
- 💳 Интеграция с ЮKassa для оплаты
- 📅 Запись на пробные тренировки
- 🔔 Автоматические уведомления
- 👨‍💼 Админ-панель для тренеров
- 📊 CRM система

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Скопируйте `.env.example` в `.env` и заполните переменные
4. Запустите бота: `python main.py`

## Структура проекта
```
├── main.py              # Главный файл запуска
├── bot/                 # Основная логика бота
├── database/            # Работа с базой данных
├── handlers/            # Обработчики команд
├── keyboards/           # Клавиатуры для бота
├── utils/               # Вспомогательные функции
└── config.py           # Конфигурация
```
