# Инструкция по настройке файла .env

Создайте файл `.env` в корне проекта и заполните следующие параметры:

## 1. 🤖 Токен бота (ОБЯЗАТЕЛЬНО!)

```
BOT_TOKEN=ваш_токен_здесь
```

**Как получить токен:**
1. Идите к [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Введите название: `СК Алекс - Тренировки`
4. Введите username: `sk_alex_training_bot`
5. Скопируйте полученный токен

**Пример токена:** `1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ123456789`

## 2. 💳 ЮKassa (для приема платежей)

```
YOOKASSA_SHOP_ID=ваш_shop_id
YOOKASSA_SECRET_KEY=ваш_secret_key
```

**Как получить данные ЮKassa:**
1. Зарегистрируйтесь на [yookassa.ru](https://yookassa.ru/)
2. Создайте магазин для вашего ИП/ООО
3. Перейдите в раздел "Настройки" → "Данные для разработчиков"
4. Скопируйте `Идентификатор магазина` и `Секретный ключ`

## 3. 👨‍💼 Администраторы

```
ADMIN_IDS=ваш_telegram_id,второй_admin_id
```

**Как узнать свой Telegram ID:**
1. Напишите боту [@userinfobot](https://t.me/userinfobot)
2. Он отправит ваш ID (например: `123456789`)
3. Если админов несколько - указывайте через запятую без пробелов

## 4. 💾 База данных

```
DATABASE_URL=sqlite:///training_bot.db
```

Оставьте как есть - используется SQLite (файловая база данных).

## 📝 Полный пример файла .env:

```env
# Конфигурация бота для СК «Алекс»

BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ123456789
YOOKASSA_SHOP_ID=123456
YOOKASSA_SECRET_KEY=live_abcdefghijklmnopqrstuvwxyz123456789
ADMIN_IDS=123456789,987654321
DATABASE_URL=sqlite:///training_bot.db
```

## ⚠️ ВАЖНО:

- **НЕ делитесь токеном бота** и секретными ключами
- **НЕ загружайте файл .env** в публичные репозитории
- Проверьте, что файл `.env` добавлен в `.gitignore`

## 🚀 После настройки:

1. Сохраните файл `.env`
2. Запустите бота: `python main.py`
3. Проверьте, что бот отвечает в Telegram
